### [Objects](../) > isBoolean

```js
isBoolean(value)
```

> Checks if <code>value</code> is a boolean value.

> Returns true is 'value' is boolean, else returns false.

- <code>value {\*}</code>

The value to check.

#### Example
```js
isBoolean(true); // => true
isBoolean(false); // => true
isBoolean({ a: 'a' }); // => false
isBoolean(null); // => false
```
