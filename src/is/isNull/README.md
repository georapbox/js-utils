### [is](../) > isNull

```js
isNull(value)
```

> Checks if <code>value</code> is null.

> Returns true is 'value' is null, else returns false.

- <code>value {\*}</code>

The value to check.

#### Example
```js
isNull(null); // => true
isNull(true); // => false
isNull({ a: 'a' }); // => false
isNull(); // => false
```
