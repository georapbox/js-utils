### [is](../) > isEvenNum

```js
isEvenNum(value)
```

Checks if a number is even.

#### Arguments

- `value` _(number)_: The number to check.

#### Returns

_(boolean)_: Returns `true` if number is even, else returns `false`.

#### Examples
```js
isEvenNum(0);
// -> true

isEvenNum(2);
// -> true

isEvenNum(3);
// -> false

isEvenNum(5);
// -> false
```
