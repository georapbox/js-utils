### [is](../) > isFalse

```js
isFalse(value)
```

Checks if `value` is false (strict equality).

#### Arguments

- `value` _(*)_: The value to check.

#### Returns

_(boolean)_: Returns `true` if value is false, else returns `false`.

#### Examples
```js
isFalse(false);
// -> true

isFalse(true);
// -> false

isFalse();
// -> false
```
