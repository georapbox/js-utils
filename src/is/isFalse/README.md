### [is](../) > isFalse

```js
isFalse(value)
```

> Checks if <code>value</code> is false (strict equality).

> Returns true is 'value' is false, else returns false.

- <code>value {\*}</code>

The value to check.

#### Example
```js
isFalse(false); // => true
isFalse(true); // => false
isFalse(); // => false
```
