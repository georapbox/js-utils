### [is](../) > isTrue

```js
isTrue(value)
```

> Checks if <code>value</code> is true (strict equality).

> Returns true is 'value' is true, else returns false.

- <code>value {\*}</code>

The value to check.

#### Example
```js
isTrue(true); // => true
isTrue(false); // => false
isTrue(); // => false
```
