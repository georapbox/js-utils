### [is](../) > isOddNum

```js
isOddNum(value)
```

Checks if a number is odd.

#### Arguments

- `value` _(number)_: The value to check.

#### Returns

_(boolean)_: Returns `true` if value is an odd number, else returns `false`.

#### Examples
```js
isOddNum(0);
// -> false

isOddNum(2);
// -> false

isOddNum(3);
// -> true

isOddNum(5);
// -> true
```
