### [is](../) > isExisty

```js
isExisty(value)
```

> Checks if <code>value</code> is existy (Not null or undefined).

> Returns true is 'value' is existy, else returns false.

- <code>value {\*}</code>

The value to check.

#### Example
```js
isExisty('Foo'); // => true
isExisty(null); // => false
isExisty(); // => false
```
