### [Arrays](../) > reverse

```js
reverse(array)
```

> Reverses an array (not in place).
> The first array element becomes the last and the last becomes the first.

- `array {Array}`

The array to reverse.

#### Example
```js
var numbers = [1, 2, 3];

var reversedNumbers = reverse(numbers); 
// -> [3, 2, 1]
```
