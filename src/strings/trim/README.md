### [Strings](../) > trim

```js
trim(str)
```

> Removes whitespace from both ends of a string.

- <code>string {string}</code>

The string to be trimmed

#### Example
```js
var str = '    Lorem ipsum dolor si amet.    ';
var trimmed = trim(str); // Lorem ipsum dolor si amet.
```
