## isEven(value) â‡’ <code>Boolean</code>
Checks if a value is even.

**Returns**: <code>Boolean</code> - True if number is even, else false.  

| Param | Type | Description |
| --- | --- | --- |
| value | <code>Number</code> | The value to check. |

**Example**  
```js
isEven(0);
// => true

isEven(2);
// => true

isEven(3);
// => false

isEven(5);
// => false

isEven('2');
// => true

isEven(null);
// => false

isEven();
// => false
```
