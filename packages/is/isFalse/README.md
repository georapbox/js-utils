## isFalse(value) â‡’ <code>Boolean</code>
Checks if a value is `false` (strict equality).

**Returns**: <code>Boolean</code> - True if the 'value' is false, else false.  

| Param | Type | Description |
| --- | --- | --- |
| value | <code>\*</code> | The value to check. |

**Example**  
```js
isFalse(false);
// => true

isFalse(true);
// => false

isFalse();
// => false
```
