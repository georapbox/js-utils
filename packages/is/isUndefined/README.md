## isUndefined(value) â‡’ <code>Boolean</code>
Checks if 'value' is undefined.

**Returns**: <code>Boolean</code> - True if the 'value' is undefined, else false.  

| Param | Type | Description |
| --- | --- | --- |
| value | <code>\*</code> | The value to check. |

**Example**  
```js
var foo;

isUndefined(foo);
// -> true

isUndefined(null);
// -> false

isUndefined({foo: 'bar'});
// -> false

isUndefined();
// -> true
```
