## isTrue(value) â‡’ <code>Boolean</code>
Checks if 'value' is true (strict equality).

**Returns**: <code>Boolean</code> - True if the 'value' is true, else false.  

| Param | Type | Description |
| --- | --- | --- |
| value | <code>\*</code> | The value to check. |

**Example**  
```js
isTrue(true);
// -> true

isTrue(false);
// -> false

isTrue();
// -> false
```
