## isOdd(value) â‡’ <code>Boolean</code>
Checks if a value is odd.

**Returns**: <code>Boolean</code> - True if value is odd, else false.  

| Param | Type | Description |
| --- | --- | --- |
| value | <code>Number</code> | The value to check. |

**Example**  
```js
isOdd(0);
// => false

isOdd(2);
// => false

isOdd(3);
// => true

isOdd(5);
// => true

isOdd('5');
// => true

isOdd(null);
// => false

isOdd();
// => false
```
